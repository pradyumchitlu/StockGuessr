================================================================================
API ENDPOINTS (12 Total)
================================================================================

Authentication (3):
  POST   /api/auth/register        Register new account
  POST   /api/auth/login           Login & get JWT token
  GET    /api/auth/me              Get current user (protected)

Matches (6):
  POST   /api/matches              Create new match (protected)
  GET    /api/matches/:id          Get match details (protected)
  PUT    /api/matches/:id          Update match results (protected)
  GET    /api/matches/history/:id  Get user's match history (protected)
  PATCH  /api/matches/:id/note     Add personal note (protected)
  DELETE /api/matches/:id          Delete match (protected)

Stock Scenarios (3):
  GET    /api/scenarios/random     Get random scenario for matchmaking
  GET    /api/scenarios            List all available scenarios
  GET    /api/scenarios/:id        Get specific scenario details

QUICK START
================================================================================

1. Install Dependencies:
   cd server && npm install
   cd ../client && npm install

2. Configure Environment:
   Copy .env.example to .env in both directories
   Add your MongoDB URI and OpenAI API key

3. Seed Database:
   cd server && npm run seed

4. Start Development:
   Terminal 1: cd server && npm run dev
   Terminal 2: cd client && npm run dev

5. Open Application:
   http://localhost:5173
